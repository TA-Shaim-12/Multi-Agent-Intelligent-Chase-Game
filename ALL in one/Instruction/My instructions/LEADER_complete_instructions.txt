PART 5 - HOW TO EXTRACT CODE FROM .ipynb FILES
================================================================

This is the most important step. Each member's code is inside a
.ipynb notebook file. You need to extract it and put it into the
correct position in the final .py file.

METHOD A - Automatic (easiest, recommended):
  Run this Python script to extract all code from a notebook:

    import json

    def extract_code(ipynb_path):
        nb = json.load(open(ipynb_path))
        code_lines = []
        for cell in nb['cells']:
            if cell['cell_type'] == 'code':
                code_lines.append(''.join(cell['source']))
        return '\n\n'.join(code_lines)

    # Usage:
    m1_code = extract_code('member1_notebook.ipynb')
    print(m1_code)

  This gives you the raw Python code from the notebook.

METHOD B - Manual (if automatic fails):
  Open the .ipynb file in VS Code
  You will see code blocks (cells) clearly separated
  Copy the code from each cell manually
  Paste it into the right position in ai_chase_game_final.py
