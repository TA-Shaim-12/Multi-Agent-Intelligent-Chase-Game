TASK 1 — Add comments to astar_path function
  Find the function called astar_path
  Add a comment above every line inside it explaining what it does
  Key things to mention in your comments:
    - A* uses the formula: f(n) = g(n) + h(n)
    - g(n) is the actual cost from the start to the current cell
    - h(n) is the estimated distance from current cell to goal
    - h(n) uses Manhattan distance: abs(row diff) + abs(col diff)
    - A* is the default algorithm the police officers use
    - It is both fast and finds the optimal path

TASK 2 — Add comments to hill_path function
  Find the function called hill_path
  Add a comment above every line inside it explaining what it does
  Key things to mention in your comments:
    - Hill Climbing always moves to the neighbour closest to the goal
    - It only looks at immediate neighbours, not the full map
    - When it gets stuck it picks a random neighbour to escape
    - It is fast but NOT guaranteed to find a path in mazes or dead ends
